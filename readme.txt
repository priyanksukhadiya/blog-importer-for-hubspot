Import and sync blog posts from HubSpot to your WordPress site with automated scheduling and comprehensive logging.

== Description ==

Blog Importer for HubSpot is a powerful WordPress plugin that seamlessly integrates your HubSpot blog content with your WordPress website. Import individual posts or set up automated synchronization to keep your content up-to-date.

**Key Features:**

* **Easy HubSpot Integration** - Connect using your HubSpot API key
* **Flexible Import Options** - Import as posts, pages, or custom post types
* **Automated Scheduling** - Set up cron jobs for regular content synchronization
* **Comprehensive Logging** - Track all import activities with detailed logs
* **Duplicate Prevention** - Smart detection prevents duplicate content
* **Content Preservation** - Maintains original formatting and metadata
* **WordPress Standards** - Fully compliant with WordPress coding standards

**Perfect for:**
* Marketing teams managing content across platforms
* Businesses centralizing their blog content
* Developers building integrated content workflows
* Anyone wanting to sync HubSpot blogs with WordPress

== Installation ==

1. Upload the `blog-importer-for-hubspot` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Blog Importer for HubSpot' in the WordPress admin menu
4. Configure your HubSpot API key and import settings
5. Start importing your blog content!

== Frequently Asked Questions ==

= Do I need a HubSpot API key? =

Yes, you'll need a HubSpot API key to connect to your HubSpot account. You can generate one in your HubSpot account settings.

= Can I schedule automatic imports? =

Absolutely! The plugin includes built-in cron scheduling to automatically sync your content at regular intervals.

= Will this create duplicate posts? =

No, the plugin includes smart duplicate detection to prevent importing the same content multiple times.

= What post types are supported? =

You can import content as posts, pages, or any custom post type available on your WordPress site.

= Can I see what was imported? =

Yes, the plugin includes comprehensive logging so you can track all import activities and troubleshoot any issues.
